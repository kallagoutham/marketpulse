How the Pieces Connect

The Django app acts as the secure application layer around this data pipeline.

  1. Stock rows are read from a CSV dataset.
  2. The producer simulates market events and publishes messages to Kafka.
  3. The consumer reads messages and writes event files into S3.
  4. Glue crawls S3 and creates catalog metadata.
  5. Athena queries the cataloged S3 data with SQL.
